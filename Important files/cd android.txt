Perfect — since you’re likely on a Xiaomi/MIUI device (most common for this INSTALL_FAILED_USER_RESTRICTED error), here’s a step-by-step guide to allow Gradle to install debug APKs via USB.

Step 1 — Enable Developer Options

Go to Settings → About phone → MIUI version.

Tap 7 times on MIUI version until you see “You are now a developer!”.

Step 2 — Enable USB Debugging

Go to Settings → Additional settings → Developer options.

Turn on USB debugging.

Step 3 — Enable “Install via USB”

In Developer options, find Install via USB.

Turn it ON.

MIUI may ask you to login to your Mi Account to allow it. Do it.

This allows Gradle / adb to install debug APKs.

Step 4 — Enable “USB debugging (Security settings)”

In Developer options, find USB debugging (Security settings).

Turn it ON.

This allows the app to be installed via USB even if it requests permissions.

Step 5 — Uninstall old app

On your phone, uninstall Nukkad completely.

Sometimes MIUI blocks install if the signing key differs.

Step 6 — Connect phone and allow permissions

Plug phone into PC via USB.

You may see a prompt “Allow USB debugging?” → Tap Allow.

You may see “Allow install via USB?” → Tap Allow.

Step 7 — Run installDebug

In your project folder:

cd C:\nukkad-chat\android
.\gradlew clean
.\gradlew installDebug


This time, Gradle should successfully install the app on the phone.

If prompted on phone, tap Allow.

✅ After this, you can run the Test Incoming Call (debug button or adb intent) and see the caller name on lock screen.
